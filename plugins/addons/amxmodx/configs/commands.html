The following commands are available:
*
amx_menu ................. = Displays the client commands menu.
amx_help ................ .. = To view the Amx Mod X commands available.
say /voteban .............. = Ban a player.
amx_langmenu ........... = To change the language.
say buyparachute ....... = Buy for falls.
amx_createnpc .......... = Creates a surfboard.
say /mp3 ................... = Opens the Music menu.
open_jail ................... = Open the cage in surf maps.
war3menu ................. = Displays Warcraft commands.
say /help .................... = Displays help on a Powered MOD, Ex: Arena Mod, Mod Warcrft, etc ...
say /hlmp ................... = Opens radios menu.
say /nextmap ............. = Display the next map.
say /thetime ............... = Displays the current time.
say /stats ................... = See stats of other players on the server.
say currentmap .......... = Displays the current map.
say /rank ................... = Displays your rank on the server.
say /top15 ................. = Displays the Top 15 of the server.
say /regras ................. = Displays the rules of the server.
amx_fakec4 ............... = Plant a fake c4, and detonates it.
bind command button .... = shortcut on a button.
*
In the console, if the admin has enabled.
